# kenjitheman.dev

## Installation

```sh
git clone https://github.com/kenjitheman/kenjitheman.dev
```

- To install all dependencies:

```sh
yarn install
```

## Usage

- To run:

```sh
yarn start
```

- To run in dev mode:

```sh
yarn dev
```
